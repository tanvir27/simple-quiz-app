1. Difference Between Map(), find(), filter() and forEach () Method

* The map() is used to transform an array by applying a function to each element in the array and returning a new array with the transformed values. It returns a new array and he original array remains unchanged.
* The filter() is used to create a new array with all the elements that pass a certain condition specified by a callback function. But It returns a new array with the filtered values.
* When we use find method it find the first element in an array that satisfies a certain condition specified by a callback function. It returns a value by checking condition. If the condition is true and found several element then it returns first element otherwise it returns undefined.
* The forEach () is used to loop over an array. It execute a callback function on each element in the array. ForEach() does not return anything. But it can modify the original array.
